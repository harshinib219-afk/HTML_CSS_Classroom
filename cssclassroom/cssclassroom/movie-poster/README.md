# 🎬 My Favorite Movies - Poster Gallery

A simple and responsive web page showcasing my favorite animated movies using **HTML** and **CSS**.

## 📌 Project Overview

This project displays movie posters in a clean and attractive grid layout. It includes hover effects and is fully responsive.

## 🚀 Features

- Clean and modern UI design
- Responsive layout using Flexbox
- Hover animations on movie cards
- Google Fonts integration (Poppins)
- Displays movie name, rating, and genre

## 🛠️ Tech Stack

- HTML5
- CSS3
- Google Fonts (Poppins)
